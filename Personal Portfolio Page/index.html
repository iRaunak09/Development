Skills

Java

Java is a high-level, class-based, object-oriented programming language designed to be highly portable. Its core philosophy, "Write Once, Run Anywhere" (WORA), means Java code is compiled into bytecode and executed by the Java Virtual Machine (JVM), allowing it to run on any device without recompilation.

HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure the content of web pages. It acts as the foundational skeleton of the internet, directing web browsers on how to display text, images, links, and multimedia.

CSS

CSS (Cascading Style Sheets) is the standard stylesheet language used to design, style, and lay out web pages. While HTML builds the basic structural skeleton of a website, CSS applies the visual presentation—including colors, typography, spacing, and multi-device responsiveness.
